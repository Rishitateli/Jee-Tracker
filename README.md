# Jee-Tracker
 JEE Study Tracker.
This project is a simple JEE Study Tracker that helps students to organize Physics,Chemistry and Mathematics chapters and their JEE preparation in simple and structured way.
I created this project to make chapter-wise planning and progress tracking easier. It helps JEE aspirants to stay organized, monitor completed topics, and identify the chapters that are still need to be studied.
Chapter-wise organization for Physics,Chemistry and Mathematics. It also has countdown timer to the number of days remaining until the target date.
This project is developed using HTML,CSS and Javascript.
I started this project to solve my own problem. Keeping track of JEE prep across three subjects felt overwhelming without a system. Building it taught me localStorage for saving progress. DOM manipulation for the collapsible sections, and how to design a UI Id which I'd actually want to use everyday. It also gave me my first real taste of what it means to build a software for a genuine need, not just as a project.
To everyone grinding for JEE - small consistent steps really do add up.
Good luck!🍀🌸
